【uTools Markdown 预览】 - 高效、专业、丝滑的预览神器

一款为 uTools 打造的高效 Markdown / HTML / SVG 预览插件，提供极致的阅读与分享体验。

✨ 核心亮点：
1. 【全格式支持】支持 Markdown、HTML 及 SVG 文件的极速渲染预览。
2. 【剪贴板预览】支持直接读取剪贴板中的文本内容并即时预览。
3. 【一键文件预览】支持在资源管理器复制 .md、.html 或 .svg 文件后直接唤起插件。
4. 【图表支持】完美支持 Mermaid 引擎（流程图、时序图、甘特图、类图、状态图、饼图等）。
5. 【高清导出】支持将预览内容一键渲染为高清图片（直接保存或复制到剪贴板），并支持导出独立样式的 HTML 文件。
6. 【灵动工具栏】底部悬浮中心设计，磨砂玻璃质感，支持智能目录跳转（TOC）。
7. 【深色模式】深度适配 uTools 主题，昼夜切换顺滑如丝。

🛠️ 快捷用法：
1. 复制内容或文件：
   - 文本：在任何地方复制你想预览的源码。
   - 文件：在资源管理器复制 .md、.html 或 .svg 后缀的文件。
2. 呼出插件：在 uTools 搜索框输入 "md预览" 或 "markdown预览"。如果是文件，uTools 会自动匹配。
3. 预览管理：
   - 支持同时开启多个预览窗口。
   - 主界面提供“显示原预览”、“从剪贴板更新”和“打开新窗口”选项。

💡 性能提示：
插件针对日常阅读场景进行了深度优化，采用渲染隔离与 GPU 加速技术，确保交互流畅、加载迅速。
