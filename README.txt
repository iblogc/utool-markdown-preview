【uTools Markdown 预览】 - 高效、专业、丝滑的预览神器

本插件为开发者、写作者精心打造，提供极致的 Markdown 阅读与分享体验。

✨ 核心亮点：
1. 【极速渲染】采用高效解析引擎，确保 Markdown 内容即刻呈现。
2. 【文件预览】支持复制本地 .md 文件后直接唤起插件进行预览（NEW!）。
3. 【图表支持】完美支持 Mermaid 引擎（流程图、时序图、甘特图、状态图、饼图等）。
4. 【灵动工具栏】分类配色设计，支持智能目录跳转（TOC），阅读无干扰。
5. 【高清导出】支持一键渲染为高清图片（直接保存或复制到剪贴板），支持导出独立 HTML。
6. 【深色模式】深度适配 uTools 主题，昼夜切换顺滑如丝。

🛠️ 快捷用法：
1. 复制内容或文件：
   - 文本：直接复制 Markdown 源码。
   - 文件：在资源管理器复制 .md 后缀文件。
2. 呼出插件：在 uTools 搜索框输入 "md预览" 或 "markdown预览"。如果是文件，uTools 会自动推荐。
3. 交互管理：
   - 支持同时开启多个预览窗口。
   - 已开启窗口时，主界面支持“更新内容”或“置顶显示”。

💡 性能提示：
插件针对日常 Markdown 阅读场景进行了深度优化，确保交互流畅、加载迅速。
