# Markdown 预览器 - 在线版

一个功能强大的在线 Markdown/HTML/SVG 预览工具，复刻了 uTools 插件的所有核心功能，并采用了独特的视觉设计。

## ✨ 功能特性

### 🎨 视觉设计
- **独特美学**：采用"Brutalist Noir"深色主题和"Industrial Minimalism"浅色主题
- **动态主题**：一键切换深色/浅色模式，自适应 Mermaid 图表主题
- **流畅动画**：精心设计的过渡效果和微交互

### 📝 编辑功能
- **多格式支持**：Markdown、HTML、SVG 三种内容类型
- **实时预览**：300ms 防抖的即时渲染
- **智能缩进**：Tab/Shift+Tab 快速调整代码缩进
- **自动保存**：每5秒自动保存，防止内容丢失

### 🔍 预览功能
- **GitHub 风格**：标准的 Markdown 渲染样式
- **代码高亮**：支持多种编程语言语法高亮
- **Mermaid 图表**：完整支持流程图、时序图、甘特图等
- **智能目录**：自动生成可折叠的文档目录

### 📤 导出功能
- **Markdown 导出**：下载/复制源码
- **HTML 导出**：生成完整的独立 HTML 文件
- **图片导出**：高清 PNG 截图，支持下载和复制到剪贴板
- **样式保持**：导出内容完美保持原有样式

### ⌨️ 快捷键
- `Ctrl/Cmd + S`：保存提示
- `Ctrl/Cmd + D`：下载 Markdown
- `Ctrl/Cmd + Shift + C`：复制 Markdown
- `Ctrl/Cmd + T`：切换主题
- `Ctrl/Cmd + B`：切换目录显示

## 🚀 使用方法

1. **直接使用**：在浏览器中打开 `index.html` 即可开始使用
2. **输入内容**：在左侧编辑器中输入 Markdown、HTML 或 SVG 内容
3. **实时预览**：右侧会实时显示渲染结果
4. **导出分享**：使用底部浮动工具栏进行各种导出操作

## 🛠️ 技术栈

- **前端框架**：原生 JavaScript (ES6+)
- **Markdown 解析**：Marked.js
- **代码高亮**：Highlight.js
- **图表渲染**：Mermaid.js
- **截图功能**：html2canvas
- **字体**：Inter + JetBrains Mono
- **样式**：CSS3 + CSS Variables

## 📱 响应式设计

- **桌面端**：左右分栏布局，最佳体验
- **平板端**：上下分栏布局，保持功能完整
- **移动端**：优化的紧凑布局，触控友好

## 🎯 设计理念

### 视觉特色
- **避免 AI 套路**：拒绝紫色渐变、Inter 字体滥用等常见 AI 设计模式

### 用户体验
- **零学习成本**：直观的界面设计，无需说明即可上手
- **高效工作流**：快捷键支持，批量操作，自动保存
- **细节打磨**：悬停效果、加载动画、错误提示等细节体验

## 🔧 本地开发

由于使用了现代浏览器 API（如 Clipboard API），建议通过本地服务器运行：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`
